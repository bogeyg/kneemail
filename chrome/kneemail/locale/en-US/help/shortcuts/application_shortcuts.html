Application Shortcuts

This Kneemail Desktop is designed to be as easy as possible to use. One great feature are the many shortcuts available to you, the user. You have more than one way to access each feature, whether by a menu item, toolbar button or a keyboard shortcut. Below are the many keyboard shortcuts available to you.

Toggle Offline/Online Mode Control+O
Close Kneemail Desktop Control+Q
View Fullscreen F11
Toggle Open/Close Sidebar Control+S
New Folder Shift+N
Delete Folder Shift+D
Edit Folder Shift+E
Print Folder Shift+P
Empty Inactive Folder Alt+I
Empty Deleted Folder Alt+D
New Entry Control+N
Delete Entry Control+D
Edit Entry Control+E
Print Entry Control+P
Search Entries Control+F
This Help System F1
