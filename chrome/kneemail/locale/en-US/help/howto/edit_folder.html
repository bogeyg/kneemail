Edit Folder
Below is a screenshot of the Edit Folder dialog:




With this dialog, you can edit a folder. You first supply a new name for the folder.

The folder "weight" is used if you have selected "Sort by Weight" in the preferences. The larger the number, the heavier its weight. The heavier the weight, the further down the folder tree the folder will appear.

When you are finished, clicking the OK button will update the folder tree to reflect the changes.
