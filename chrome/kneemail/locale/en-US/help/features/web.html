Web Resources
Web resources are integrated into this software. Anywhere from searching the Bible to pointing the integrated Web browser to your favorite site can be done from the Web Resources screen. There is a multi-engine search box available to you if you want to do a quick search without leaving this application. The included engines are:
  • Kneemail's Web Search
  • Bible Gateway
  • CrossMap
  • WikiChristian
  • Got Questions?
  • SeekFind.org
  • Answers in Genesis
Although the integrated Web browser does allow you to access any Web Site on the Internet, it is not designed to be a replacement for your favorite browser. However, the basics, such as forward and back buttons, are included.
