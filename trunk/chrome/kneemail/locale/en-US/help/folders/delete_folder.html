Delete Folder

The following are the ways to delete the selected folder:

  • Clicking this toolbar button.
  • Choosing this action from the Folder menu.
  • Choosing this action from the Folder context menu (right-click menu).
  • Pressing Shift+D on the keyboard.

Below is a screenshot of the confirmation prompt:

Once you have clicked OK, the folder, its entries and subfolders will be moved the the Deleted Folder. Once you have emptied the deleted folder, the folder, entries and subfolders will not be recoverable.
