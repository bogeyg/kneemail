Entries
Entries are your prayer requests, praises and general thoughts. This topic will teach you how to create, edit and delete entries.

The following are ways to create an entry:
  • From the file menu, choose New Entry.
  • While the mouse curser is over the entries list, from context menu (usually the right mouse button), choose New Entry.
  • Click the toolbar button.
  • The shortcut key Control-N.
The following are ways to edit an entry:
  • From the file menu, choose Edit Selected Entry.
  • While the mouse curser is over the entries list, from context menu (usually the right mouse button), choose Edit Selected Entry.
  • Double click the entry.
  • Click the toolbar button.
  • The shortcut key Control-E.
The following are ways to delete an entry:
  • From the file menu, choose Delete Selected Entry(ies).
  • While the mouse curser is over the entries list, from context menu (usually the right mouse button), choose Delete Selected Entry(ies).
  • Click the toolbar button.
  • The shortcut key Control-D.
The following are ways to mark a prayer entry "Answered":
  • From the file menu, choose Mark the Selected Entry Answered.
  • While the mouse curser is over the entries list, from context menu (usually the right mouse button), choose Mark the Selected Entry Answered.
  • Click the toolbar button.
  • The shortcut key Control-A.
The following is the way to mark an entry "Inactive":
  • From the compose screen, change it's status to "Inactive."
