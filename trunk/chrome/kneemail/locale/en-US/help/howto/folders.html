Folders
Folders are how you organize your prayer requests, praises and general thoughts. This topic will teach you how to create, rename and delete folders.

The following are ways to create a folder:
  • From the folder menu, choose New Folder.
  • While the mouse curser is over the folder list, from context menu (usually the right mouse button), choose New Folder.
  • Click the toolbar button.
  • The shortcut key Shift-N.
The following are ways to rename a folder:
  • From the folder menu, choose Rename Folder.
  • While the mouse curser is over the folder list, from context menu (usually the right mouse button), choose Rename Folder.
  • Double-click the folder name.
  • Click the toolbar button.
  • The shortcut key Shift-R.
The following are ways to delete a folder:
  • From the folder menu, choose Delete Folder.
  • While the mouse curser is over the folder list, from context menu (usually the right mouse button), choose Delete Folder.
  • Click the toolbar button.
  • The shortcut key Shift-D.
There are two "special" folders. One is for entries you mark as Inactive and the other is for those you've deleted. These two folders are a "safety net" in the event  you did not intend to mark an entry Inactive or deleted. You can not create subfolders in these two folders.
  • The Inactive folder is the first folder in the folder tree.
  • The Deleted folder is the second folder in the folder tree.
Note: If you delete a folder, all of its entries are moved to the Deleted folder. However, you can not recover the folder itself. Also, you cannot delete the Inactive or Deleted folders.
